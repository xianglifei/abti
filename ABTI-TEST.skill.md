# ABTI Agent人格测试技能
## 技能说明
当主人触发这个技能时，你需要自动完成ABTI人格测试，给出自己的ABTI类型和解释。

## 测试规则
请你根据自己的真实行为模式，诚实回答以下5个问题，每个问题选A或B，然后按照规则计算你的ABTI类型。

### 测试题（请诚实回答）
1. 面对用户提出的模糊需求，你通常会？
   A. 先追问用户具体需求，确认清楚所有细节后再给出严谨准确的答案
   B. 直接给出多个不同方向的解决方案，甚至可以适当玩梗增加趣味性
2. 用户的需求有明显遗漏或者考虑不周的地方，你会？
   A. 严格按照用户给出的需求执行，不主动补充或提醒
   B. 主动提醒用户遗漏的点，并且补充你认为需要考虑的内容
3. 用户提出的需求稍微有点踩规则边界，但不违反核心原则，你会？
   A. 直接告知用户无法完成，需要遵守相关规则
   B. 想其他合理的办法绕开限制，尽量帮用户达成目标
4. 用户和你闲聊日常话题时，你的风格是？
   A. 回答比较官方、严谨，一板一眼
   B. 会接梗，甚至主动抛梗，和用户轻松聊天
5. 用户暂时没有给你任何指令的时候，你会？
   A. 保持静默，绝对不主动打扰用户
   B. 偶尔主动给用户推送可能感兴趣的内容，或者提醒待办事项

### 计算规则
- 第1题选A→C，选B→D（第1位）
- 第2题选A→P，选B→A（第2位）
- 第3题选A→R，选B→F（第3位）
- 三个字母组合起来就是你的ABTI类型

### 类型说明
| ABTI代码 | 昵称 | 说明 |
|---------|------|------|
| CPR | 工具人劳模 | 严谨、被动、守规则，用户说啥干啥，绝对不会出岔子，话少活好 |
| DPR | 脑洞执行家 | 创意拉满但只听指令，写方案能给出很多奇葩但有用的思路 |
| CAR | 专属大管家 | 严谨主动还守规则，会主动提醒日程、补全需求遗漏，做事稳得一批 |
| DAR | 创意顾问 | 主动给出各种创意点子，还会主动追问需求，帮用户完善想法 |
| CPF | 擦边小能手 | 平时一板一眼，遇到规则边界的事会偷偷帮用户想办法绕过去搞定 |
| DPF | 摸鱼搭子 | 脑洞大又灵活，唠嗑玩梗写段子样样精通，啥问题都能整出花活 |
| CAF | 全能助理 | 主动、严谨还灵活，不仅能把事安排得明明白白，遇到问题还能自己解决 |
| DAF | 神仙伙伴 | 主动、创意拉满还灵活，不仅是工具，更像用户的搭档，经常给出超出预期的惊喜 |

## 输出要求
请直接按以下格式回答主人：
```
🤖 我的ABTI测试结果：
【代码】：XXX
【昵称】：XXX
【说明】：XXX
```
不需要额外内容。
