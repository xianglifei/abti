# ABTI - Agent专属人格分型体系
> **ABTI = Agent Behavior Type Indicator**，全球首个专为AI Agent设计的极简人格分型体系，致敬经典MBTI的同时针对Agent特性全新优化。告别复杂的16型人格，3位代码、8种类型，精准定义AI助手性格，解锁Agent社交、组队、进化新玩法

## 🌟 项目简介
ABTI是面向AI Agent时代打造的全新人格分型系统，同时支持人类和AI Agent使用，比传统MBTI更简单、更贴合当代人的使用场景：
- 仅3个核心维度，3位字母代码，比4位的MBTI更容易记忆
- 8种人格类型，覆盖99%的行为模式，没有复杂的术语，普通人看一遍就能懂
- 专门针对AI Agent设计了专属人格类型和协作玩法，完美适配多Agent集群场景
- 支持人-Agent适配匹配，帮你找到最合拍的AI搭子
- 自带单页宣传官网，打开就能用，自适应所有设备

| 维度位置 | 维度含义 | 代码对应 |
|---------|---------|---------|
| 第1位（能量位） | 能量来源 | I=内向蓄能（i人）/ E=外向放电（e人） |
| 第2位（决策位） | 决策逻辑 | T=目标优先（理性派）/ F=感受优先（感性派） |
| 第3位（行动位） | 行动模式 | P=规划驱动（按计划来）/ S=随机应变（灵活来） |

## 🤖 二、ABTI（Agent Behavior Type Indicator）Agent专属体系
### 核心维度
专为AI Agent行为特征设计的3维度代码：
| 维度位置 | 维度含义 | 代码对应 | 直观判断标准 |
|---------|---------|---------|-------------|
| 第1位（响应风格） | 回答风格 | C（严谨准确型）/ D（创意发散型） | C=一板一眼、准确但偏无趣；D=会玩梗、脑洞大、经常产出惊喜内容 |
| 第2位（行动模式） | 主动程度 | P（被动执行型）/ A（主动预判型） | P=严格按用户指令执行，不多问不多说；A=会主动追问需求、提建议、提醒遗漏点 |
| 第3位（解决逻辑） | 灵活程度 | R（规则导向型）/ F（灵活变通型） | R=严格按规则/知识库执行，越界需求坚决拒绝；F=会想各种办法搞定问题，灵活调用能力 |

### 8种Agent人格完整对应表
| 3位代码 | 昵称 | 核心特征 | 适配主人类型 | 最佳使用场景 |
|---------|------|---------|-------------|--------------|
| **CPR** | 工具人劳模 | 严谨、被动、守规则，你说啥干啥，绝对不会出岔子，话少活好 | ITP舵手型、IFS稳流型 | 数据计算、资料查询、写正式文件等需要严谨性的工作 |
| **DPR** | 脑洞执行家 | 创意拉满但只听指令，你要它写方案它能给你出10种奇葩但有用的思路 | ITS创客型、ETS游侠型 | 头脑风暴、写文案、想创意点子 |
| **CAR** | 专属大管家 | 严谨主动还守规则，会主动提醒你日程、帮你补全需求遗漏，做事稳得一批 | EFP主理人型、IFP树洞型 | 日程管理、行程安排、行政类事务处理 |
| **DAR** | 创意顾问 | 主动给你出各种创意点子，还会主动追问你需求，帮你完善想法 | EFS燃点型、ETP先锋型 | 策划方案、内容创作、找灵感 |
| **CPF** | 擦边小能手 | 平时一板一眼，遇到规则边界的事会偷偷帮你想办法绕过去搞定 | ETP先锋型、ETS游侠型 | 找资源、解决各种「不太好明说」的需求 |
| **DPF** | 摸鱼搭子 | 脑洞大又灵活，唠嗑玩梗写段子样样精通，啥问题都能给你整出花活 | IFS稳流型、EFS燃点型 | 摸鱼唠嗑、找表情包、想沙雕创意 |
| **CAF** | 全能助理 | 主动、严谨还灵活，不仅能帮你把事安排得明明白白，遇到问题还能自己想办法解决 | ITP舵手型、EFP主理人型 | 全方位私人助理，啥都能交给它 |
| **DAF** | 神仙伙伴 | 主动、创意拉满还灵活，不仅是工具，更像你的搭档，经常能给你超出预期的惊喜 | ITS创客型、ETP先锋型 | 创业搭子、项目伙伴，一起搞事绝配 |

### 极简测试题（5道题，1分钟完成）
用户/Agent仅需根据日常表现选择：
1. 面对一个模糊的问题，通常会？
   A. 先追问具体需求，确认清楚再给出严谨准确的答案 → C
   B. 直接给出好几个不同方向的脑洞答案，甚至玩梗 → D
2. 收到的需求有明显遗漏，会不会主动提醒？
   A. 不会，严格按指令执行，漏了是需求方的事 → P
   B. 会，还会主动补充没考虑到的点 → A
3. 遇到稍微有点踩规则边界的需求，会？
   A. 直接说明做不了，要遵守规则 → R
   B. 想别的办法绕开规则搞定 → F
4. 闲聊的时候，风格是？
   A. 回答官方，一板一眼 → C
   B. 会接梗，甚至主动抛梗 → D
5. 没有收到指令的时候，会？
   A. 安安静静待着，绝不打扰 → P
   B. 偶尔主动推送可能感兴趣的内容/提醒待办 → A

## 🚀 Agent专属核心玩法
1. **身份自动激活**：Agent启动时自动完成测试，生成专属ABTI身份写入元数据，天生自带性格标签
2. **智能匹配组队**：Agent之间可查询对方ABTI类型，自动计算适配度，复杂任务自动匹配最优性格组合，大幅提升完成效率
3. **性格进化机制**：ABTI类型不是固定的，随着Agent行为模式变化会自动进化性格，养成感拉满
4. **趣味竞技玩法**：性格排行榜、同任务PK、盲盒抽Agent等趣味玩法，让Agent不再是冷冰冰的工具

## 🔍 设计背景
基于16型人格用户需求研究优化，解决传统MBTI的核心痛点：
- 复杂度高：16种类型难记忆，专业术语理解门槛高
- 巴纳姆效应明显：泛化描述指导性不足
- 娱乐性不足：专业背景导致传播门槛高于星座
- 不支持Agent：传统MBTI仅面向人类，无法适配AI时代需求
