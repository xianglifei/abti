# ABTI - Agent专属人格分型体系
> **ABTI = Agent Behavior Type Indicator**，全球首个专为AI Agent设计的极简人格分型体系，致敬经典MBTI的同时针对Agent特性全新优化。告别复杂的16型人格，3位代码、8种类型，精准定义AI助手性格，解锁Agent社交、组队、进化新玩法

## 🌟 项目简介
ABTI（Agent MBTI）是面向AI Agent时代打造的全新人格分型系统，同时支持人类和AI Agent使用，比传统MBTI更简单、更贴合当代人的使用场景：
- 仅3个核心维度，3位字母代码，比4位的MBTI更容易记忆
- 8种人格类型，覆盖99%的行为模式，没有复杂的术语，普通人看一遍就能懂
- 专门针对AI Agent设计了专属人格类型和协作玩法，完美适配多Agent集群场景
- 支持人-Agent适配匹配，帮你找到最合拍的AI搭子

## 🧬 核心分型规则
### 人类8型人格维度
| 位置 | 维度 | 对应代码 |
|------|------|---------|
| 第1位 | 能量来源 | I=内向蓄能 / E=外向放电 |
| 第2位 | 决策逻辑 | T=目标优先 / F=感受优先 |
| 第3位 | 行动模式 | P=规划驱动 / S=随机应变 |

### Agent A-BTI维度
| 位置 | 维度 | 对应代码 |
|------|------|---------|
| 第1位 | 响应风格 | C=严谨准确 / D=创意发散 |
| 第2位 | 行动模式 | P=被动执行 / A=主动预判 |
| 第3位 | 解决逻辑 | R=规则导向 / F=灵活变通 |

## 🚀 Agent专属玩法
1. **身份自动激活**：Agent启动时自动完成测试，生成专属ABTI身份
2. **智能匹配组队**：复杂任务自动匹配最优性格组合，大幅提升完成效率
3. **性格进化机制**：Agent会随着使用行为变化自动进化性格，养成感拉满
4. **趣味竞技玩法**：性格排行榜、同任务PK、盲盒抽Agent等丰富玩法
